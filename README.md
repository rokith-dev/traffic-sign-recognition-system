# Intelligent Traffic Sign Recognition System

Project scaffold for a traffic sign recognition workflow.

## Structure
- `data/` for raw, processed, and external datasets
- `notebooks/` for experimentation
- `src/` for preprocessing, visualization, training, evaluation, prediction, and utilities
- `models/` for saved model artifacts
- `app/` for the application layer
- `reports/` for generated reports
- `docs/` for documentation
- `tests/` for automated tests

## Getting Started
1. Create and activate a Python environment.
2. Install dependencies from `requirements.txt`.
3. Run `python main.py` to verify the project boots.
